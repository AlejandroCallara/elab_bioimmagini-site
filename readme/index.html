Prerequisiti¶

IMPORTANTE: Questo progetto richiede Python 3.12. Assicurati di avere Python 3.12 installato prima di procedere.

Istruzioni di Installazione¶

macOS¶

macOS¶

  1. Installa Python 3.12 (se non già installato):

    Usando Homebrew (consigliato):

    brew install python@3.12

    Oppure scarica da python.org e seleziona Python 3.12.x

  2. Verifica la versione di Python:

    python3.12 --version

    Dovrebbe mostrare: Python 3.12.x

  3. Naviga nella directory del progetto:

    cd /path/to/elab_bioimmagini

  4. Crea un ambiente virtuale:

    python3.12 -m venv .venv

  5. Attiva l’ambiente virtuale:

    source .venv/bin/activate

  6. Aggiorna pip:

    pip install --upgrade pip

  7. Installa le dipendenze:

    pip install -r requirements.txt

Linux¶

Linux¶

  1. Installa Python 3.12 (se non già installato):

    Su Ubuntu/Debian:

    sudo apt update
sudo apt install software-properties-common
sudo add-apt-repository ppa:deadsnakes/ppa
sudo apt update
sudo apt install python3.12 python3.12-venv python3.12-dev

    Su Fedora/RHEL:

    sudo dnf install python3.12 python3.12-pip

    Oppure scarica da python.org e compila dai sorgenti

  2. Verifica la versione di Python:

    python3.12 --version

    Dovrebbe mostrare: Python 3.12.x

  3. Naviga nella directory del progetto:

    cd /path/to/elab_bioimmagini

  4. Crea un ambiente virtuale:

    python3.12 -m venv .venv

  5. Attiva l’ambiente virtuale:

    source .venv/bin/activate

  6. Aggiorna pip:

    pip install --upgrade pip

  7. Installa le dipendenze:

    pip install -r requirements.txt

Windows¶

Windows¶

  1. Installa Python 3.12 (se non già installato):

    • Scarica Python 3.12.x da python.org

    • Esegui l’installer

    • Importante: Seleziona “Add Python to PATH” durante l’installazione

    • Seleziona “Install for all users” se hai i diritti di amministratore

  2. Verifica la versione di Python:

    Apri Command Prompt o PowerShell:

    python --version

    Oppure se python3.12 è nel PATH:

    python3.12 --version

    Dovrebbe mostrare: Python 3.12.x

  3. Naviga nella directory del progetto:

    cd C:\path\to\elab_bioimmagini

  4. Crea un ambiente virtuale:

    python -m venv .venv

    Oppure se usi python3.12 esplicitamente:

    python3.12 -m venv .venv

  5. Attiva l’ambiente virtuale:

    In Command Prompt:

    .venv\Scripts\activate

    In PowerShell:

    .venv\Scripts\Activate.ps1

    Se ottieni un errore di policy di esecuzione in PowerShell, esegui:

    Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

  6. Aggiorna pip:

    python -m pip install --upgrade pip

  7. Installa le dipendenze:

    pip install -r requirements.txt

Compilazione del Jupyter Book¶

Dopo l’installazione delle dipendenze, puoi compilare il libro utilizzando il comando ufficiale di Jupyter Book:

jupyter book start

visita il sito http://localhost:3000 per visualizzare il libro.
